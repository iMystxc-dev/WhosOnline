# WhosOnline

WhosOnline is a plugin used to display the heads of online players in a GUI

Preview: https://gyazo.com/732fb06638acab661e9a596cc433d1a7

## Installation

Requirements:
- Pixelmon Reforged
- GooeyLibs 2.1.0

## Usage

```yml
Commands:
- /online

Permissions:
- whosonline.command.online
```
